# AEG-LAB — 電動ガン カスタムシミュレーター

パーツ構成から **サイクル → ピストン圧縮 → 銃口初速 → 弾道** までを一気通貫で数値計算する、単一HTMLのウェブアプリです。ビルド不要・依存ライブラリなし。`index.html` をブラウザで開くか、GitHub Pages にそのまま配置して動きます。

姉妹プロジェクト: [bb-ballistics](https://adspawn.github.io/bb-ballistics/)（外部弾道コアを本シミュレーターの Stage 4 に移植しています）

---

## 計算の4ステージ

| Stage | 内容 | モデル |
|---|---|---|
| 1 | サイクル・レスポンス・消費電流 | DCモーター線形負荷特性（無負荷回転数×失速トルク）＋バッテリー内部抵抗・回路抵抗による電圧サグの反復収束。コッキング時のシリンダー吸気負圧損失を含む |
| 2 | ピストン運動・空気圧縮 | **2室モデル**（シリンダー室⇄ノズルオリフィス⇄バレル室）。dt=1µs の数値積分。各室は質量M・内部エネルギーUを追跡し P=(γ−1)U/V でエネルギー整合 |
| 3 | バレル内加速・銃口初速 | Stage 2 と同一ループ内。BB-バレル隙間からのブローバイ漏れを含む |
| 4 | 外部弾道 | Morrison / Loth (2021) 抗力係数、マグヌス揚力 (Cl=0.12)、回転減衰 (Cf=1.328/√Re)。dt=0.1ms |

### 物理モデルの要点

- **サン＝ヴナン式の圧縮性流れ** — ノズル流路と BB 隙間漏れの両方に適用。圧力比が臨界値 1.893 を超えると流速が音速で頭打ちになる**チョーク現象**を再現します（ハイレートスプリング＋絞りノズルで発生）。
- **ピストンヘッド一方向弁（バウンスポンピング）** — 実ピストンヘッドのOリングは前進時に密閉・後退時に開放されます。この弁効率（詳細パネル `ピストンヘッド弁効率`、既定 0.85）をモデル化したことで、実測と一致する重量弾特性（純正構成で E(0.43g)/E(0.20g) ≈ 1.0）が物理から自然に導かれます。0 にすると完全密閉ヘッド相当（比 ≈ 0.8）まで低下します。
- **吸気負圧** — コッキング時、吸気オリフィス（既定 φ8mm）が後退速度に追いつかないとシリンダー内が負圧になりモーター負荷が増えます。30rps 級ハイサイクルで吸気を絞ると ROF が明確に低下します。
- **ジュールクリープ（流速チューン）** — ノズル絞り単体は絞り損失で初速・エネルギーを**下げます**。流速チューンの本質は「絞り＋強スプリング」の組み合わせで、0.20g のエネルギーを法定上限に合わせたまま重量弾のエネルギーを引き上げることです。本モデルでの検証例（0.20g=0.90J に揃えた場合）:

  | ノズル径 | 必要スプリング | 0.43g エネルギー | クリープ比 |
  |---|---|---|---|
  | 4.3mm（標準） | M128相当 | 1.05 J | 1.17 |
  | 3.2mm（絞り） | M139相当 | 1.17 J | 1.30 |
  | 2.6mm（流速） | M153相当 | 1.29 J | 1.43 |

  ※このため「重量弾特性」カードは 0.43g 使用時に 0.98J を超える構成へ警告を出します。**重量弾でも必ず実測してください。**

### 校正済みの代表値（既定の物性値にて）

- M100・バレル300mm・4/5シリンダー・0.20g → 約92 m/s / 0.85 J
- 東京マルイ ハイサイクル近似プリセット → 約81 m/s / 23 rps（レビュー実測 80–87 m/s・最大25 rps と整合）
- DSG プリセット（8歯・M140・2発/回転）→ 約42 rps、レスポンス約20 ms

## 機能

- **パーツ選択**: バッテリー / モーター（ブラシ付き・ブラシレス）/ 配線・FET / ギア比 / セクター（SSG・**DSG**）/ スプリング / ピストン重量 / **ショートストローク（歯数カット）** / **AOEパッド厚** / シリンダー加速ポート / **ノズル・流路オリフィス径** / バレル長・内径 / BB重量・直径 / ホップ
- **全項目カスタム入力** — 各パーツの「カスタム」チップから任意値を直接入力可能。一般許容値を逸脱した構成のシミュレートに対応
- **詳細物理定数パネル** — ストローク、シリンダー内径、プリロード、死容積、摩擦、ギア効率、漏れ係数、ノズルCd、吸気径、ホップ保持力、弁効率、空気密度を直接編集
- **一般版 / プロ版モード** — 用途別かんたん構成と、実測校正・安全診断・詳細グラフのフル機能
- **実測校正** — 実銃のクロノ値・RPS を入力すると計算値へのスケール補正を適用
- **安全診断** — 電池Cレート超過、ESC電流域、**BLDC過電圧**、PME注意、DSG設定不整合、容積比異常、重量弾0.98J超過を自動警告
- **構成図** — Ver.2メカボックス断面がパーツ選択に連動（スプリング巻き密度・ポート位置・ノズル径・バレル長・AOEパッド・実効ストローク表示）。「発射」でサイクルタイムに応じた動作アニメーション
- **グラフ** — 内部弾道（ピストン位置・シリンダー/バレル内圧・BB弾速の時系列）と弾道図（ホップあり/なし・ゼロイン・射線）、距離別減衰テーブル

## このリビジョンでの修正（査読結果）

物理コアを全構成でヘッドレス検証した結果、**計算式・数値は正しく動作**していることを確認しました。その上で以下を修正しています。

1. **スマホでグラフが縦長化するバグを根治** — `setupCanvas` が `height` 属性を読み→ `cv.height = h×dpr` で**同じ属性を上書き**していたため、高DPI端末（dpr=2〜3）では再描画のたびに高さが dpr 倍に増殖していました（PCは dpr=1 のため無症状）。設計高さを `data-*` に退避し、CSS表示高さを明示指定する方式に変更。狭幅では高さ・軸目盛りも自動調整します。
2. **リサイズ時の再計算を軽量化** — スマホのアドレスバー伸縮による resize でフル物理計算（内部弾道×5回＋ホップ二分探索）が走っていたのを、グラフ再描画のみに変更。
3. **流速プリセットの副作用バグ** — `PHYS.hopHold=2.4` が他プリセットへ切り替えても残留していたのを、プリセット適用時に既定値へ復帰＋詳細パネル表示へ反映するよう修正。
4. **DSG のレスポンス値** — 1発あたり半回転で係合するため `responseMs` を発数で割るよう修正（約41ms→約20ms）。
5. **ブラシレス過電圧の警告を追加** — 定格11.1VのBLDCに14.8V等を選ぶと安全診断に表示。
6. **立体断面機能を削除**（ご要望）— トグルボタン・3D描画分岐・関連状態を除去。
7. 歯数カット注記を実装値（72mm/16歯 = 1歯4.5mm）に整合。

### モデルの限界（README必読事項）

- モーター・スプリング等の物性値は代表的な市販パーツの**近似値**です。個体差・電池の劣化・気密・給弾抵抗・温度は含みません。
- ブラシレスは「8.4V換算の線形トルク特性＋ESC電流目安」という簡易モデルです。実際のESCの電流制限・進角制御は再現しません。
- DSG はコッキング仕事とROFの補正のみで、タペットタイミング・給弾追従・PME（プリエンゲージメント）は対象外です（診断で注意喚起）。
- **銃口エネルギー 0.98 J 以下（日本国内法）を、使用する弾でクロノグラフ実測のうえ厳守してください。** 本シミュレーターの数値は保証値ではありません。
